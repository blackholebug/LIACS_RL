The name of the folder is of the form "board6-20s-ab\_with\_idtt-mct\_10000", 
which in order tells :
the board size, 
the number of total matches that two players played, 
player alpha-beta is equipped with transposion table or not, 
and finally the number of MCTS player's number of rollout. 

In each folder you can see 4 graphs 
with the name in the form of "player1-player2" (player1 means who is the first one to play).
